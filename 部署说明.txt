方式1: 快速部署（推荐）
──────────────────────────────────────────────────────────
1. 打开 QUICK_DEPLOY.md 查看3步部署流程
2. 或者运行自动化脚本（需先编辑 GitHub 用户名）：
   
   .\deploy.ps1

方式2: 使用 GitHub Desktop（最简单）
──────────────────────────────────────────────────────────
1. 下载 GitHub Desktop: https://desktop.github.com/
2. 登录您的 GitHub 账户
3. File → Add Local Repository
4. 选择: H:\material project\官方文档
5. Publish Repository
6. 仓库名: materials-project-api-docs
7. 点击 Publish

方式3: 手动命令行
──────────────────────────────────────────────────────────
在 PowerShell 中执行：

cd "H:\material project\官方文档"

# 第1步：在 GitHub 网站创建新仓库
# 访问 https://github.com/new
# 仓库名: materials-project-api-docs
# 不要勾选 "Add a README file"

# 第2步：添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/materials-project-api-docs.git

# 第3步：推送代码
git branch -M main
git push -u origin main

完成！访问您的仓库查看效果。
